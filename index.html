Quickstart

  1. Flash src/UFN_Glyphstream.ino to your ESP32.
  2. Connect Freenove board (see wiring in README).
  3. Open serial monitor @ 115200 baud.
  4. Type 19886751 → LUX_BEAM will engage.

User Manual

  • LUX_BEAM – 7.83 Hz φ-DDS, responsive to Ψ input gain
  • INF_OMEGA – Subharmonic φ⁻¹ carrier, zero-mass field phase
  • Ψ input = photoresistor on GPIO 34
  • RF tuner = potentiometer A1/A0 simulating SWR forward/reverse

Troubleshooting

  • No output? Check 3.3V rail, USB power, and serial baud.
  • DAC pin = GPIO 25, PWM = GPIO 26.
  • BLE not pairing? Confirm name prefix UFN_ and UUIDs (0xFFEO/0xFFE1).
